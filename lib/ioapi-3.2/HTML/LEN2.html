LEN2()

Fortran version: 

    INTEGER FUNCTION LEN2( J1, J2, BUFFER )
        INTEGER      , INTENT(IN   ) :: J1
        INTEGER      , INTENT(IN   ) :: J2
        CHARACTER*(*), INTENT(IN   ) :: BUFFER

NO C version:

Summary:

Returns the number of leading blanks in the substring BUFFER( J1:J2 ) of the Fortran character string argument BUFFER.

Superseded by LBLANK(). See also TRIMLEN()

Preconditions:

none

Fortran Usage: 

    ...
    CHARACTER*256  BUFFER
    INTEGER        L, M
    INTEGER        LEN2, TRIMLEN
    ...
    L = 1 + LEN2( 1, 256, BUFFER ) )
    M = TRIMLEN( BUFFER ) )
    WRITE (*,*) 
  &  'The effective contents of BUFFER is "', 
  &  BUFFER( L:M ) , '"'
    ...

C Usage:

don't
